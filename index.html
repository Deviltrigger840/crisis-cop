Research Framework:

Literature Survey

This project enhances slave robots' adaptability by integrating external environmental data (e.g., gas, humidity, pressure) with internal health metrics (e.g., voltage, temperature) for proactive maintenance and real-time response. Unlike past studies, it introduces short-term environmental forecasting and dynamic power optimization based on environmental stressors. Combining sensor fusion with traction control and alert systems, it creates a holistic, adaptive robot system for resilient operation in diverse and changing warehouse conditions.

Overview

Aimed to design an autonomous outdoor patrol robot that can travel in the diverse terrain and weather conditions while being very reliable. In operation, real-time environmental analysis has adapted to these physical systems. These are designed for various groups including security teams, facility managers, or organizations seeking scalable surveillance solutions. The challenges in multi-sensor fusion algorithms involve the handling of missing and noisy data. It implements a strong algorithm that can make meaningful insights even from incomplete or unreliable data, hence enhancing the ability of the robots to make accurate environmental predictions for optimum performance. For now, the functionalities are being simulated with python.

Problem Statement

Traditional patrol systems struggle with adaptability, decision-making, and coordination in dynamic environments. This causes inefficiencies in navigation, monitoring, and resource use. Research Objective

  • Develop an energy-efficient algorithm that minimizes power consumption while ensuring real-time monitoring and secure data transmission through edge computing.
  • Design a robotic system with Lidar, and GPS that patrols the warehouses autonomously. The route should automatically be updated according to environmental conditions.
  • A sensor-based system for slave robots enabling adaptive weather and AQI prediction, fault detection, and traction control. It uses real-time IoT data and a web dashboard to ensure reliable performance and autonomous decision-making.
  • An algorithm enabling real-time communication and coordination among robots, reducing latency and response time. It also autonomously detects incidents and dynamically adjusts patrol routes for enhanced coverage.

Methodology

The system combines Reinforcement Learning for adaptive path planning, LIDAR-SLAM for real-time mapping and obstacle avoidance, and GPS-waypoint navigation for precise routing. Onboard sensors monitor environmental and health data, processed via machine learning for predictive insights. Edge computing and IoT enable real-time communication, while a distributed coordination system ensures efficient, autonomous collaboration among slave robots.
